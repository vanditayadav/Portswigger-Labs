# Authentication Labs

This section contains my hands-on learning and practical work from PortSwigger Authentication labs.

Authentication is the process of verifying a user's identity before granting access to an application or system. Authentication vulnerabilities occur when flaws in this process allow attackers to bypass security controls, gain unauthorized access, or compromise user accounts.

Through these labs, I explore common authentication weaknesses and understand how insecure implementations can introduce security risks.

Topics explored include:

- Username Enumeration
- Brute-force related issues
- Password reset vulnerabilities
- Multi-factor authentication weaknesses
- Authentication bypass techniques
